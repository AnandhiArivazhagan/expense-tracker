# expense-tracker
MERN stack application to track daily expenses of a person 
Tech Stack: HTML5, CSS3, JavaScript, Chart.js, Node.js, Express.js, MongoDB
•	Developed a full-stack expense tracking app where users can set monthly salary, log daily/monthly expenses, and monitor remaining budget in real time.
•	Implemented CRUD operations for expense management — add, edit, and delete transactions with persistent MongoDB storage.
•	Integrated Chart.js to render interactive pie charts showing spending distribution across categories for visual financial insights.
•	Built an Excel export feature allowing users to download their complete expense history for offline tracking and record-keeping
